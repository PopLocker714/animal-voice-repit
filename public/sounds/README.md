# Звуки животных (CC0)

Сюда кладём аудиофайлы животных. Имена должны совпадать с `src/data/animals.ts`:

| Файл | Животное |
|------|----------|
| `sheep.mp3` | Овца 🐑 |
| `cow.mp3` | Корова 🐄 |
| `cat.mp3` | Кошка 🐱 |
| `dog.mp3` | Собака 🐶 |
| `rooster.mp3` | Петух 🐓 |
| `duck.mp3` | Утка 🦆 |
| `goat.mp3` | Коза 🐐 |
| `pig.mp3` | Свинья 🐷 |

## Где взять (всё CC0 / общественное достояние)

- **Pixabay Sound Effects** — https://pixabay.com/sound-effects/search/animal/ (лицензия Pixabay, без атрибуции)
- **Freesound** — https://freesound.org/ (фильтр по лицензии Creative Commons 0)
- **BBC Sound Effects** — https://sound-effects.bbcrewind.co.uk/ (личное использование)

Скачай короткие (1–3 сек) чистые записи, переименуй по таблице, положи сюда.
Приложение работает и без файлов — просто покажет «звук ещё не добавлен».
